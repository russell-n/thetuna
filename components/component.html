The Base Components¶

Introduction¶

This is a module to hold the base class that (loosely) implements the Composite Pattern. See the Tuna Plugin for how the Composites are being used in the tuna (it is implemented in the Tuna-plugin’s product property).

The BaseComponent Class¶

This is the base-class which the other classes will inherit from. All Plugin products should look like this (it doesn’t do much but every method is called at some point by the Composites so all plugin products should implement it).
